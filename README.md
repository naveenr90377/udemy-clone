# 🎓 Udemy Clone

A simple Udemy homepage clone built using HTML and CSS.

## 🚀 Features
- Responsive layout using Flexbox
- Course cards section
- Categories with hover effect
- Topics and popular courses section
- Trusted companies section
- Clean footer design

## 🛠️ Tech Used
- HTML
- CSS

## 📂 Folder Structure
- index.html
- style.css
- images/

## 🎯 Purpose
This project was created to practice front-end development and UI design.
